# homebrew-spm

Homebrew tap for **[spm](https://github.com/hiadamhere/spm)** — the Skill and Plugin Manager, a CLI
that installs AI-agent skills from Git catalogs into your coding agents.

## Install

```sh
brew install hiadamhere/spm/spm
```

Or tap first, then install:

```sh
brew tap hiadamhere/spm
brew install spm
```

Upgrade with `brew upgrade spm`. Works on macOS (Apple Silicon + Intel) and Linux.

## What's here

A single formula, `Formula/spm.rb`, that downloads the self-contained `spm` binary for your platform
from the [main repo's releases](https://github.com/hiadamhere/spm/releases) and pins its checksum — no
.NET required.

## Note

`Formula/spm.rb` is **generated by spm's release pipeline** on every version tag and pushed here
automatically by CI — **please don't edit it by hand.** Issues, feature requests, and PRs about spm
itself belong on the **[main repo](https://github.com/hiadamhere/spm)**.
